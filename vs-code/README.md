### About

Collection of extensions and further tips for [VS Code](https://code.visualstudio.com/).

### VS Code Extensions

#### Extensions that must to be installed for better development experience

* Python
* Pylance
* Jupyter
* autoDocstring
* indent-rainbow
* Code Spell Checker
* Docker
* Dev Containers
* Remove - SSH
* Remote Development
